## Default Permission

Default permissions for axum plugin

#### This default permission set includes the following:

- `allow-call`
- `allow-call-json`
- `allow-fetch`
- `allow-fetch-send`
- `allow-fetch-cancel`
- `allow-fetch-read-body`
